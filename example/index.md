Index
=====

Welcome to the example page, this website will quickly let you see all features
of *kursg* program in action.

If you look into source of the file (**index.md**) you will notice meta field
order:

~~~
 [kursg-meta]: # (order: 1)
~~~

Thanks to this, we can keep our index site at the very first place in the menu

.kursg-ignore
=============

Note, that in example there is directory *sub5* which is copy of *sub4*, but
*sub5* contains *.kursg-ignore* file, and thus *sub5* and everything below will
be ignored by *kursg*
